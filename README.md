# Marketing Pulse AI Dashboard

Dashboard inteligente para análise de campanhas de marketing digital com IA integrada.

## Funcionalidades

- **Dashboard em Tempo Real** - Métricas atualizadas automaticamente
- **Análise de IA** - Insights inteligentes e otimizações automáticas
- **Gráficos Interativos** - Visualizações avançadas com Chart.js
- **Gestão de Campanhas** - CRUD completo com interface intuitiva
- **Análise de Sentimento** - Processamento de comentários com IA
- **Tema Claro/Escuro** - Interface adaptável
- **Responsivo** - Funciona perfeitamente em todos os dispositivos
- **Relatórios** - Geração automática de relatórios detalhados

## Tecnologias

- **Frontend**: React 18, Chakra UI, Chart.js
- **Estado**: Context API, React Hooks
- **Estilização**: CSS3, Chakra UI System
- **Ícones**: React Icons
- **Gráficos**: Chart.js, React-Chartjs-2

## Como Executar

### Pré-requisitos
- Node.js 16+
- npm ou yarn

## 📋 **Sobre o Projeto**

O **Marketing Pulse AI Dashboard** é uma Single Page Application (SPA) desenvolvida em React que simula um dashboard profissional para análise de campanhas de marketing digital. O projeto demonstra habilidades avançadas em:

- 🎯 **Desenvolvimento Frontend Moderno**
- 🎨 **Design de Interface e UX**
- 📊 **Visualização de Dados**
- 🔧 **Arquitetura de Componentes**
- 📱 **Responsividade e Acessibilidade**

---

## ✨ **Funcionalidades**

### 🏠 **Dashboard Principal**
- 📊 Métricas em tempo real de campanhas
- 📈 Gráficos interativos de performance
- 🤖 Simulação de insights gerados por IA
- 🔔 Sistema de notificações inteligentes
- 📋 Previsões de alcance e ROI

### 🎯 **Gestão de Campanhas**
- ➕ Criação de campanhas com wizard intuitivo
- 🔍 Filtros avançados por status e plataforma
- ⏯️ Controles de play/pause para campanhas
- 📊 Métricas detalhadas por campanha
- 🗑️ Gerenciamento completo do ciclo de vida

### ⚙️ **Configurações**
- 👤 Gestão de perfil do usuário
- 🔔 Configurações de notificações
- 🎨 Personalização de interface
- 🔒 Configurações de segurança
- 🌐 Opções de idioma e fuso horário

### 🎨 **Interface e UX**
- 📱 Design 100% responsivo
- ✨ Animações suaves e micro-interações
- 🌙 Suporte a modo escuro (preparado)
- ♿ Acessibilidade web (WCAG)
- 🚀 Performance otimizada
